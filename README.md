# Rnday Event Website

This is the official website for **Rnday**, an event organized by 2nd-year [CMI](https://reseau-figure.fr/) students at the UFR-ST of Besan√ßon. The event's theme is **"Quantum Computers: The Future of Computer Science?"**. The website was built using the template [TheEvent - v2.2.0](https://bootstrapmade.com/theevent-conference-event-bootstrap-template/), with custom HTML, CSS, and JavaScript.

## Features

- **Event Details**: Provides information on the schedule, speakers, and venue.
- **Access to event**: Allows attendees to access the event hosted on https://www.gather.town/
- **Access to activities**: 3 differents are available, all about quantum computing:
  - Two labyrinths
  - An animation
  - An escape game like video game

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/rnday-event.git
   ```
2. **Open `index.html`** in a web browser to view the website.
